Epoch 와 Batch Size 에 대해 간단명료하게 설명을 드리자면,

Epoch : 전체 데이터를 몇 번 반복해서 볼 건가요 ? -> 여러 번 보면 더 좋으니까.
Batch size : 데이터를 몇 개씩 볼 건가요 ? -> 나눠서 보면 편하니까.

이외에도 Learning rate, Initial Weight 가 있고 이를 Hyperparameter 라고 한다.
